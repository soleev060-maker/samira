background-color — цвет фона

background-color: red;
background-color: #ff0000;
#ff0000 — это HEX
HEX — это шестнадцатеричный код цвета.
#RRGGBB
• RR — красный
• GG — зелёный
• BB — синий
#ff0000 значит:
• ff (максимум красного)
• 00 (нет зелёного)
• 00 (нет синего)
Получается красный.

background-color: rgb(255, 0, 0);
RGB — это Red, Green, Blue

background-color: rgba(255, 0, 0, 0.5);
• 255 - красный
• 0 - зелёный
• 0 - синий
• 0.5 - прозрачность

background-image — картинка фона
background-repeat — повтор фона

background-position — положение фона
background-position: center;
background-position: top;
background-position: bottom right;

background-size — размер фона
background-size: cover;
background-size: contain;
background-size: 200px;
contain - фон полностью умещается, может остаться пустое место
cover - фон полностью закрывает блок, может обрезаться

background-attachment — движение фона
background-attachment: scroll;
background-attachment: fixed;
scroll - фон движется вместе со страницей
fixed - фон остаётся на месте, текст двигается

form — форма
Форма нужна, чтобы:
• отправлять данные
• вводить текст
• делать регистрацию

action — куда отправляются данные
action указывается внутри (form).

label — подпись к полю
label — это текст, который объясняет, что вводить.
Пример: (label)Имя(/label) (input type="text")
for — связь label с input
(label for="name")Имя(/label)
(input type="text" id="name")
type — тип поля ввода
type используется в (input)
Пример: (input type="text")

div — контейнер (блок)
div — это пустой блок-контейнер.
Сам по себе ничего не делает, а нужен, чтобы:
• группировать элементы
• задавать стиль (цвет, размер, фон)
• строить структуру страницы

outline — это внешняя линия вокруг элемента,
но она отличается от border.
Главное отличие:
• border занимает место
• outline НЕ занимает место и не сдвигает элементы
outline-style — стиль обводки
Определяет, какая будет линия.
